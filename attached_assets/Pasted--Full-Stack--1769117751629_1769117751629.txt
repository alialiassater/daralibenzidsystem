أنت مهندس برمجيات Full Stack ومحلل نظم محترف.

المطلوب: إنشاء تطبيق ويب متكامل لإدارة مطبعة ودار نشر، بواجهة عربية 100% مع دعم الباركود.

⚠️ شروط أساسية:
- لغة الواجهة: العربية فقط
- اتجاه الصفحة: من اليمين إلى اليسار (RTL)
- جميع الأزرار، الرسائل، الجداول، والتنبيهات باللغة العربية

التقنيات:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: SQLite أو PostgreSQL
- Barcode Library: JsBarcode أو QuaggaJS

====================
1️⃣ المستخدمون والصلاحيات
====================
- تسجيل دخول عربي
- مدير النظام (صلاحيات كاملة)
- موظف (إضافة وتعديل فقط)

====================
2️⃣ إدارة المخزون + باركود
====================
- صفحة "المخزون"
- إضافة مادة جديدة:
  • اسم المادة
  • النوع (ورق، حبر، غلاف، كتاب...)
  • الكمية
  • السعر
- توليد باركود تلقائي لكل مادة
- طباعة الباركود (PDF)
- مسح الباركود بالكاميرا أو إدخاله يدوياً
- عند المسح:
  • جلب المادة تلقائياً
  • تسجيل خروج/دخول
- تنبيه عند انخفاض المخزون
- سجل كامل لحركة المخزون

====================
3️⃣ إدارة طلبات الطباعة
====================
- إضافة طلب طباعة
- اسم الزبون
- نوع الطباعة
- عدد النسخ
- نوع الورق
- التكلفة
- حالة الطلب
- ربط الطلب بالمواد المستعملة عبر الباركود

====================
4️⃣ إدارة دار النشر
====================
- صفحة "الكتب"
- لكل كتاب:
  • اسم الكتاب
  • المؤلف
  • ISBN
  • باركود خاص بالكتاب
  • عدد النسخ المطبوعة
  • عدد النسخ المباعة
  • عدد النسخ المتبقية
- بيع الكتب عن طريق مسح الباركود

====================
5️⃣ المبيعات والمحاسبة
====================
- تسجيل البيع عبر الباركود
- حساب المداخيل اليومية والشهرية
- تسجيل المصاريف
- حساب الأرباح تلقائياً

====================
6️⃣ لوحة التحكم (Dashboard)
====================
- إجمالي المخزون
- عدد الطلبات
- المبيعات
- تنبيهات المخزون
- إحصائيات برسومات

====================
7️⃣ متطلبات الواجهة
====================
- تصميم نظيف وبسيط
- Responsive
- دعم RTL في CSS
- خطوط عربية
- تعليقات واضحة في الكود
